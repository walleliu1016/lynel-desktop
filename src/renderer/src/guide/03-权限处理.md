# 权限处理

Claude 执行敏感操作（运行命令、写文件等）前会请求授权。Lynel Desktop 把权限请求同步到多个入口，方便你在任何地方处理。

## 权限请求出现时

当会话进入"等待授权"状态：

- 左侧会话列表的状态标签变为**等待授权**；
- 会话标签页图标变为警示样式；
- 灵动岛（屏幕顶部的悬浮胶囊）弹出提示；
- 如果配置了企业微信通道，会同时推送一张权限卡片。

![等待授权状态](guide-images/06-awaiting-permission.png)

## 三种处理方式

任选其一处理，其余入口会自动关闭：

1. **终端内处理**：像在命令行使用 Claude 一样，用方向键选择允许/拒绝后回车。
2. **灵动岛处理**：鼠标移入灵动岛展开面板，点击允许或拒绝按钮；Claude 的提问（选择题）也可以直接在面板中作答。
3. **企业微信处理**：
   - 普通权限：点击卡片上的 **允许** / **拒绝** 按钮（模板卡片），一步完成审批。
   - 单选/多选提问：点击选项后提交，多问题会逐题发送。
   - 也可使用命令：`/allow 3` / `/deny 3` / `/answer 3 <答案>`。

![企业微信权限面板](guide-images/07-notch-permission.png)

> 在终端里自行处理后，企业微信会收到"权限已在终端处理"的通知，会话状态同步恢复为运行中。
