# 模型供应商

在 设置 → 供应商 中管理 Claude 使用的 API 供应商，支持配置多个并随时切换，适合在官方 API、代理网关或第三方兼容服务之间选择。

![供应商设置](guide-images/14-provider-settings.png)

## 添加与切换

- 点击左侧列表右上角的 **+** 新增供应商。
- 选中某个供应商后点击**设为当前**，之后新启动的会话将使用该供应商（已在运行的会话不受影响）。
- 当前生效的供应商会显示"当前"徽标。

## 配置项说明

| 配置项 | 对应环境变量 | 说明 |
|--------|--------------|------|
| Base URL | `ANTHROPIC_BASE_URL` | API 地址，官方为 `https://api.anthropic.com` |
| Auth Token | `ANTHROPIC_AUTH_TOKEN` | 鉴权令牌 |
| 默认模型 | `ANTHROPIC_MODEL` | 未细分场景时使用的模型 |
| Haiku 默认模型 | `ANTHROPIC_DEFAULT_HAIKU_MODEL` | 留空则使用默认模型 |
| Sonnet 默认模型 | `ANTHROPIC_DEFAULT_SONNET_MODEL` | 留空则使用默认模型 |
| Opus 默认模型 | `ANTHROPIC_DEFAULT_OPUS_MODEL` | 留空则使用默认模型 |
| 推理模型 | `ANTHROPIC_REASONING_MODEL` | thinking / extended thinking 场景专用，与普通模型分开配置 |

## 测试连接

填好 Base URL 后点击**测试连接**，可验证地址与令牌是否可用，再保存生效。

> 注意：删除操作不可恢复，且至少需要保留一个供应商。
