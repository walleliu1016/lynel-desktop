# 快速开始

Lynel Desktop 是 Claude Code 的桌面伴侣：用图形界面管理多个 Claude 会话，随时查看运行状态，还能通过企业微信远程收发消息和处理授权。

## 界面总览

![主界面总览](guide-images/01-overview.png)

- **标题栏**：左侧显示正在运行的会话数量；右侧依次是使用指南、设置和窗口按钮。点击关闭按钮不会退出程序，而是隐藏到系统托盘。
- **左侧会话列表**：展示所有会话及其实时状态，支持搜索和收起。
- **右侧标签页**：首页、各个会话、设置分别以标签页打开，可随时切换。

## 创建第一个会话

1. 在首页点击新建按钮，或点击会话列表顶部的"新建"。
2. 选择工作目录（Claude 将在这个目录下工作）。
3. 可以顺手填写第一条提示词，例如"帮我分析这个项目的结构"。
4. 点击创建，等待终端启动完成。

![新建会话对话框](guide-images/02-new-session.png)

## 在终端中与 Claude 对话

会话标签页内是一个完整的交互式终端，与在命令行里直接使用 Claude 完全一致：

- 直接输入文字，按回车发送。
- Claude 弹出的选择菜单、权限确认等都可以用方向键和回车操作。
- 终端尺寸会随窗口大小自动调整。

![会话终端运行中](guide-images/03-terminal.png)

> 提示：如果 `claude` 命令不在系统 PATH 中，请到 设置 → 通用 里填写 Claude 可执行文件的完整路径。
