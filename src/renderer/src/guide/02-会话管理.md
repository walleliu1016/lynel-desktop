# 会话管理

## 会话列表与状态

左侧面板实时展示所有会话，每个会话显示项目名、最近消息数和当前状态：

| 状态 | 含义 |
|------|------|
| 运行中 | Claude 正在思考、输出或执行工具 |
| 等待授权 | Claude 在等你批准一个权限请求 |
| 已完成 | 本轮回复已结束，等待下一条消息 |
| 已结束 | 会话进程已退出 |

![会话列表与状态标签](guide-images/04-session-list.png)

列表顶部支持**搜索会话**，也可以点击收起按钮把列表折叠成窄栏，给终端留出更多空间。

## 恢复历史会话

之前创建过的会话（包括直接在命令行里使用 Claude 产生的会话）都会出现在列表中。点击即可打开：Lynel Desktop 会自动以恢复模式启动 Claude，完整保留之前的对话上下文。

## 重命名与复制 ID

在会话条目的菜单中可以：

- **重命名**：给会话起一个好记的名字（默认使用首条提示词或 AI 生成的标题）。
- **复制 Session ID**：用于企业微信 `/bind` 等命令中精确指定会话。

![会话菜单](guide-images/05-session-menu.png)

## 多会话并行

每个会话是独立的标签页和独立的 Claude 进程，可以同时运行多个会话互不干扰。标签页图标会实时反映会话状态：转圈表示运行中，警示图标表示等待授权。
