## 0.0.9

### 界面

- TraceSidebar 新增收起/展开按钮，收窄后只留展开按钮，与左侧 SessionList 对称
- 面板展开收起图标改用 PanelLeftClose/Open、PanelRightClose/Open，更直观
- 面板展开收起按钮加大（Trace 28×28、SessionList 32×32），带边框和 hover 变色
- 会话列表标题改用蓝色 13px/700 字体，meta 信息提升亮度，增强视觉层级
- 右键菜单和 Bot 选择器背景加深 + 窗口阴影，悬浮感更强
- Bot 选择器每项间分割线 + hover 背景加深
- BotManagement 改为固定列宽表格布局 + 表头行，编辑表单双列排列
- TitleBar 运行中计数与品牌名对齐

### 新功能

- 推送思考过程、推送工具调用默认关闭，新安装后需手动开启
- macOS Dock 图标保证显示 + 点击 Dock 图标恢复窗口
- Trace 新请求到达时自动滚动到底部（用户翻看历史时不强制拉回）

### Bug 修复

- 会话列表不再加载全部历史会话（refreshList 只更新已有条目）
- macOS Dock 图标缺失问题
