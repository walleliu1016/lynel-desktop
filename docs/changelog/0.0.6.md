## 0.0.6

### 项目结构重构

- 对齐 Electron 标准项目结构：`electron/main.ts` → `src/main/index.ts`，`electron/preload.ts` → `src/main/preload.ts`
- 渲染进程 `frontend/` → `src/renderer/`，`tsconfig.electron.json` → `tsconfig.json`
- 更新 package.json / electron-builder.yml / CI workflow 中所有路径引用
- 清理 dist-electron、dist、.claude-flow 等临时目录

### 灵动岛修复

- 修复 AskUserQuestion 中文输入时空格导致弹窗消失（IME 选词框误触发 mouseleave）
- 有待处理权限请求时阻止 mouseleave 自动折叠
- 修复灵动岛生产环境路径层级错误

### 企业微信增强

- 命令前缀从 `#` 改为 `/`，新增 `/help` 列出全部可用命令
- 入站消息剥离群聊 @botname 前缀，确保命令检测不受 @mention 影响
- 修复通道保存时 Vue reactive Proxy 导致 structured clone 失败

### 供应商与通道设置

- 模型供应商设为默认后自动写入 settings.json（applyActiveProvider）
- 新增模型供应商测试连接功能
- 通道设置页面重构为「当前/启用/新增/删除」风格
- 模型供应商「设为当前」即时保存

### 其他改进

- PTY 退出后终端显示覆盖层与「重新进入」按钮
- 版本号从 package.json 动态读取，修复左下角用户信息缺失
- 新建 session 支持 Claude CLI 可选参数多选（--verbose/--debug）
- electron-store cwd 统一到 ~/.lynel-desktop
- 自动允许命令执行时，排除 AskUserQuestion 仍走交互流程
