## 0.0.10

### 破坏性变更

- **移除企业微信 `/` 命令**：删除所有 `/list`、`/create`、`/status`、`/to`、`/pending`、`/help`、`/allow`、`/deny`、`/answer` 等命令拦截逻辑，避免拦截 Claude 原生的 `/help`、`/compact` 等命令。权限请求和 AskUserQuestion 全部改为模板卡片交互，不再支持文字命令。删除 `docs/wecom-commands.md`，更新 `04-Bot集成.md` 使用指南

### 新功能

- F12 快捷键切换 DevTools（生产模式也可用）
- Trace 列表显示调用时间（HH:mm:ss），Overlay 宽度改为 50%
- 新增 `docs/usage.md` 使用指南，精简 README
- apiproxy 增强上游错误日志（含 upstream/path/code/syscall/hostname/port）

### Bug 修复

- macOS Dock 图标尺寸过大，改用 `app.dock.setIcon` 单独设置
- app.dock 可选链修复，避免非 macOS 下 TS 报错
- Windows 退出时递归清理 claude 进程树

### 重构

- wecom-channel 去掉冗余的 enabled 开关
