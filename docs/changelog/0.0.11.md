## 0.0.11

### 新功能

- **设置项落地**：`auto_start`（开机自启）、`log_enabled`（日志开关）、`claude_path`（自定义 Claude CLI 路径）、`prevent_sleep`（防止系统休眠，基于 `powerSaveBlocker`）
- **模型供应商模型列表拉取**：新增 `app:fetchProviderModels` IPC，调 `GET /v1/models`，兼容 OpenAI / LiteLLM / One API 格式
- **模型字段 combobox**：5 个模型字段从纯文本 input 改为可搜索下拉框，聚焦展开、输入过滤、点击选中，保留手动输入能力
- **测试连接 OpenAI 格式兼容**：先试 `/v1/messages`，404 回退 `/v1/chat/completions`，OpenAI 格式供应商给出警告（Claude CLI 需要代理）
- **智能 URL 拼接**：`base_url` 已带 `/v1` 时不再重复拼接

### Bug 修复

- **重连 session 白屏**：主进程新增终端输出环形缓冲区，重连时回放到新 xterm 实例，避免 spinner 隐藏后白屏 4-5 秒
- **loading 动效一闪而过**：`ptyExisted` 路径不再立即隐藏 spinner，改为等待 buffer 有可见内容
- **模型列表交互**：改为用户主动输入 URL/Token 才触发拉取，切换供应商不再自动触发；切换时清空 `availableModels`
- **获取模型列表 toast 提示**：成功显示数量，失败提示手动输入
- **文案修正**：「自动允许 Bash 执行」改为「自动允许工具调用」（实际控制所有非 AskUserQuestion 工具）
- **gitignore 范围**：仅排除 `src/main/` 和 `src/renderer/src/` 下的编译产物
