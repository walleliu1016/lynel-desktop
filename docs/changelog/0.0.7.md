## 0.0.7

### 使用指南

- 标题栏新增"使用指南"入口，以标签页打开，左侧章节导航
- 内置 6 章内容：快速开始、会话管理、权限处理、企业微信集成（含完整命令样例）、模型供应商、常见问题
- 配 14 张界面截图，内容宽度随窗口自适应

### 全局 Tab 与会话管理

- 全新全局 Tab 系统，多会话标签页并行切换
- 会话重命名与标题自动获取
- 设置页保持会话列表可见并支持收起
- 终端右键菜单支持复制、粘贴、全选；复制 Session ID 显示提示
- 修复会话列表滚动条缺失、时间显示错误，tooltip 改为 3 秒延迟显示

### 权限与状态同步

- 修复在终端处理权限后会话列表状态不联动刷新的问题
- 企业微信 `/allow` `/deny` `/answer` 改为会话序号定位，与 `/bind` `/to` 约定统一
- 终端自行处理权限后企业微信提示"已在终端处理"，不再显示猜测的决策
- 基于 apiproxy/hook 事件修复会话状态机，增强权限等待视觉提示

### 企业微信增强

- `/create` 支持工作目录与提示词，创建后桌面端自动打开会话
- 支持 AskUserQuestion 卡片回答、命令别名与 Markdown 消息排版
- prompt 去重避免同一条用户消息重复推送；移除 `/to` 的多余成功提示

### 主题与界面

- 红蓝主题 VS Code 风格主页与会话管理改造
- 统一三端应用图标为红蓝主题，新增 Ubuntu 支持
- 移除 OLED 暗色主题并自动迁移已保存配置到浅色
- 通用设置新增灵动岛显隐开关

### 其他修复

- 修复 spinner 启动动画卡住、通道删除不持久化、供应商配置自动填充与即时保存
- 修复退出时资源回收与临时文件清理
- 修复切换设置后终端滚动条消失
