Session Tabs Redesign

基于你截图的反馈：1) 顶部 ToolBar 与 Tab 重复且显细碎；2) Tab 不像标签页。下面给出 3 个方案。

方案 A：真 Tab 栏，详情进 Tooltip（推荐）

核心改动：去掉独立 ToolBar，把 Tab 做成真正的标签页（圆角上边框 + active Tab 与终端区同色相连）。Tab 上只保留标题、状态点、关闭按钮；完整项目/ID/创建来源等信息，hover 到 Tab 上时以 Tooltip 显示。

方案 B：卡片式 Tab + 极简标题栏

核心改动：保留一条极窄的标题栏（只显示当前 session 标题），Tab 改成更明显的小卡片（圆角矩形、active 有底色）。项目/ID 等详情 hover 标题栏或 Tab 时 tooltip 展示。

方案 C：VS Code 编辑器 Tab（顶色条）

核心改动：Tab 完全仿 VS Code 编辑器标签：active tab 顶部有一条 accent 色细线，tab 之间用竖线分隔，背景与终端区一致。同样去掉 ToolBar，详情 hover 显示。

我的建议

推荐 方案 A：它最干净地解决了你提出的两个问题——去掉重复 ToolBar、让 Tab 看起来像真正的标签页。hover tooltip 补充完整信息，不会增加常驻视觉噪音。如果你更在意“一眼就是 VS Code”，可以选方案 C。
